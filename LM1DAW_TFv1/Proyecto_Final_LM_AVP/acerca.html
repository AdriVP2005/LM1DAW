📁 Acerca del Proyecto 📁

📌 Descripción General 📌

Este proyecto consiste en una aplicación web para visualizar el clima actual utilizando HTML, CSS y JavaScript. El usuario puede consultar el estado del tiempo en su ubicación actual o una ciudad específica.

🎯 Objetivos del Desarrollo 🎯

  1. Practicar el uso de tecnologías frontend como HTML, CSS y JavaScript.
  2. Integrar datos reales a través de una API del clima.
  3. Ofrecer una interfaz intuitiva y atractiva para el usuario.

🔗 APIs 🔗

  • Al final no se utilizaron.

🛠 Tecnologías Empleadas 🛠

  • HTML5
  • CSS3
  • JavaScript
  • Bootstrap 5
